[ 개인 웹사이트 프로젝트 ]

(컨셉)
중고상품 공유 및 ‘판매’ 사이트

  1. 메인 페이지(main)
    - 검색 기능 (일반적인 검색 + 해쉬태그로 검색)
    - 최근 등록된 상품 (최근 5개의 상품 진열)
    - 관심 상품 진열 (최근 지정한 5개 진열)
    - 자유게시판

  2. 회원가입 / 로그인 페이지(register)
    - 회원가입과 로그인 페이지는 한 페이지로 통합
    - 간편 로그인 기능 지원
    - 아이디 중복 확인, 비밀번호 양식 확인(8~16자 대문자, 특수기호, 숫자 포함)
    - 휴대폰 인증 기능 지원
    - 로그인 기록 저장 기능 지원

  3. 상품 목록 페이지(list)
    - 맨 위에 검색창 고정
    - 일반 검색 해쉬태그로 검색 기능 구현
    - 상품 이미지, 제목, 판매자 이름, 가격, 즐겨찾기 기능, 개시날짜 표시
    - 상품을 누르면 해당 상품 상세 페이지로 이동
    - 작성자를 누르면 작성자 페이지로 이동 > 유저 정보 페이지

  4. 상품 상세 페이지(detail)
    - 제목 옆에 배지로 거래중, 거래완료 표시
    - 배지 옆에 즐겨찾기 기능 구현
    - 게시글 밑단에 해시태그 기능
    - 작성자를 누르면 팝업창으로 간단하게 유저표시
    - 댓글 기능 구현
    - 게시글 하단에 있는 거래하기 누르면 작성자와 채팅 연결

  5. 유저 페이지(userpage) -> 유저번호로 연결
    - 간단한 유저 정보 개시 (이름, 프로필 사진, 게시글 수 등등)

  6. 게시글 작성 페이지(write) / 수정, 삭제 기능 (작성자와 관리자 가능)
    - 제목, 이미지 첨부, 상품 내용, 가격, 해시태그
    - 제목 & 가격은 정수
    - 작성자 본인은 수정 및 삭제 기능 지원

  7. 관리자 페이지(admin)
    - 홈, 회원, 게시글, 태그, 공지 관리페이지 지원
    - 회원, 게시글, 공지 페이지에는 검색 기능 지원
    - 홈에는 일일, 주간 방문자 통계 표시


  8. 기타 기능
    - 게시글 즐겨찾기 기능
    - 공지 및 알림 기능
    - 실시간 채팅 기능
    - 우편번호 API 기능

회원 정보
  회원번호 (int) PK
  회원 등급 (varchar) FK
  이름 (varchar 20) NN
  아이디 (varchar 20) 중복불허
  비밀번호 (varchar 100) 8~16자내 특문, 대문자, 숫자 최소 1개 포함 -> 저장은 암호화
  이메일 (varchar 100)
  우편번호 (text) FK
  성별 (int) 남(0), 여(1) 둘 중 하나
  휴대폰 번호 (varchar 20) 숫자로만 받되 010으로 시작하는 숫자만
  가입일 (datetime) 자동부여
  방문 수 (int) 접속마다 증가
  거래 성공 횟수 (int) 거래완료를 누를때 마다 증가

  제목 (varchar 100) NN
  내용 (longtext) 
  첨부이미지 (??) FK
  가격 (int) 천단위마다 , 표시
  게시 날짜 (datetime) 자동부여
  조회수 (int) 조회마다 증가
